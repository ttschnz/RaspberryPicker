# Python Interface for Raspberry Picker

## Setup & Requirements
- [Python](https://www.python.org/downloads/)
- Python packages:
  - Open the terminal in this directory
  - type in `python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt`
  - hope for the best

## Running the code 
When requirements (python and packages) are met the following steps open the applicatoin:
- Open the terminal in this directory
- type in `venv\Scripts\activate && python main.py`
